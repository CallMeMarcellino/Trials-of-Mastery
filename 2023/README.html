What is Trials of Mastery?

A simple game jam I've created for me and my friends, so that we can hold ourselves accountable, improve our game development skills, and learn more about the Godot engine.

Times

  • Game submissions will be open from August 14th to December 1st.

  • Voting will continue until January 31st, but as mentioned above, you can't submit a game after December 1st.

  • All dates mentioned above, other then the end date, are at 12 PM.

  • On the other hand, the end date, is at 12 AM.


Socials

For now, this jam will just be using Excalian's personal Discord server as it's main social platform. You can give yourself access to the game jam through the "Channels & Roles", all updates will be posted into the "Misc. Updates" channel.


Voting Criteria

  • Gameplay

  • Music

  • Art

  • Polish

  • Uses Godot


Optional Theme

Life Flashing By


FAQ

  • Do I have to follow the theme?

No, the theme is completely optional. Additionally, there are no extra points for following it.

  • Do I need to use Godot?

Yes, that's the whole point of this game jam: to improve your Godot skills, so if you don't use Godot, it won't make sense.


Last Thing

The games that you submit to this jam, must be made for the jam. This jam is not a way to advertise your already-made games, it's a way to learn and improve your own skills. To help you when you create games in the future, or when your fixing games in the present.
